# CFO Task Tracker — Казатомпром

Система контроля задач и поручений по дочерним и зависимым организациям (ДЗО) Группы Казатомпром для CFO.

## Функциональность

- Список задач с фильтрацией по статусу и сортировкой по приоритету/сроку
- Недельный календарь (Outlook-style) с цветовой кодировкой приоритетов
- Навигация по 26 ДЗО и 7 департаментам
- Загрузка протоколов совещаний (.docx) с извлечением поручений
- Тёмная и светлая тема
- Сохранение данных в localStorage

## Запуск

```bash
npm install
npm run dev
```

## Стек

React 18 · TypeScript · Vite · Tailwind CSS · Zustand · date-fns · lucide-react
